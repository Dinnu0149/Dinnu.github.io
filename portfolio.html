Construction Issue Tracker

The Construction Issue Tracker is a web application that help manage project and it issues in an construction site.This project is divided into different section, we have the Construction-projects,issue,resolved-issue and category; for the listed objects i liked to call them the action in the project, while the Employee,job,department,posistion; fall under the subject.When a user register to the site either using the form or the social authentication the user instance is created and a automatic unique id is created other than the one created in the database.After an admin creates a project it is associated with the user instance and his unique id, then after vaildation it been sent to the database which now become a foreign key to the issue object.Same process happens to the issue object, it also become a foreign for the resolved object.
